# AWS 启动 Windows 11 EC2

AWS 中国区没有直接提供 Windows 11 AMI，VM Import/Export 也需要额外准备镜像和导入流程。一个折中方式是先启动 Linux EC2，再通过开源 reinstall 脚本把系统重装成 Windows。

项目地址：[bin456789/reinstall](https://github.com/bin456789/reinstall)

参考文档：[CSDN 原文](https://lpwmm.blog.csdn.net/article/details/155258680?spm=1001.2014.3001.5502)
